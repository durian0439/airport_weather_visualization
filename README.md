# airport_weather_visualization

어떤 자료를 이용해서 시각화 할 지 고민했는데, 전 직장에서 해외 공항 자료를 작성했을 때 해당 지역의 기상 자료를 활용했던 것이 생각이 났다.
공항의 일별 기상정보를 통해 어떻게 시각화를 할지 고민하고 구현해 보겠다.
